Penguin Histogram

The histogram below depicts the performance of the drunken penguin.The penguin's homework grades are divided into five bins each containing a range of five grade points. The frequency of the grades in each bin is shown by the Y-Axis.


To generate this histogram I simply created an excel sheet for all 19 of the drunken penguin's homework grades. Using excel's built-in histogram tool I created the histogram below. Then, I made modifications to fix the number and size of the bins. After, I made some visual modifications to make the histogram look more pleasing.
